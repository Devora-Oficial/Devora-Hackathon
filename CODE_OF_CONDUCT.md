#  Código de Conduta

## Nosso Compromisso

Para promover um ambiente aberto e acolhedor, nós, mantenedores e contribuidores, nos comprometemos a fazer com que a participação neste projeto seja uma experiência livre de assédio para todos.

## Padrões Esperados

Exemplos de comportamento positivo:

- Demonstrar empatia e respeito.
- Aceitar críticas construtivas com maturidade.
- Focar no que é melhor para o projeto.
- Ser amigável com novos participantes.

Exemplos de comportamento inaceitável:

- Uso de linguagem ofensiva.
- Comentários desrespeitosos.
- Assédio público ou privado.
- Qualquer conduta inadequada profissionalmente.

## Responsabilidades dos Mantenedores

- Remover, editar ou rejeitar comentários e contribuições que não sigam este código.
- Aplicar medidas apropriadas em caso de violação.

## Escopo

Este Código de Conduta se aplica dentro dos repositórios, issues, pull requests e qualquer comunidade relacionada ao projeto.

## Aplicação

Casos de violação podem ser reportados diretamente ao responsável:  
**Guilherme Nantes** – via GitHub ou outro canal disponível.

Todas as denúncias serão revisadas com confidencialidade.
